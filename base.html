Implementation:

Initialize the app with a tabview with 5 tabs: discover, posts, services, look up and you. Each tab is connected to a view that will be put on/off from the main view base on user actions. "Posts" is the default page.


Posts: A scrollable view which contains topics or paragraphs published by a registered user. Other user can step into the topic/paragraph to join the conversation. User can like, share, and comment on any topic/paragraph. (e.g. Twitter)


Discover: A multi-function page that displays real-time events such as party invitations and games.


Services: A page contains on-campus and off-campus services.


Look up: Class information, club information and personnel information.


You: User's own profile.
